**Календарь автомобильных событий по РФ – приложение для автолюбителей. Дрифт, ралли, выставки, слёты по всей России**

Основные возможности для пользователей:
- Календарь событий по России
- Фильтры: регион, тип события (дрифт/выставка/слёт/ралли)
- Избранное + экспорт в Google/Apple Calendar

Для организаторов:
- Добавление событий (с модерацией)
- Редактирование, статистика просмотров
- Ссылки на билеты/регистрацию

Технологии:
### Frontend – KMP
- Архитектура D-KMP
- Паттерн в UI слое - MVI
- voyager **screen model** для ViewModel
- voyager **navigation** для навигации
- koin для DI
- Http - ?
